**Authors:** B. Steffen, A. Simmons This is an R script for reading .xls/.csv files and plotting histogram of one column. **Key functions are:** hist(filename) - will make a histogram of your data file to run this script, download R from your prefered CRAN mirror (http://www.r-project.org/), and select Ctrl+R NOTE: to do an .xls file it requires the xlsReadWrite library More info about R graphing functions: http://www.harding.edu/fmccown/R/
